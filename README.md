# Projeto de Extensão: Gamificação Aplicada à Matemática do Ensino Fundamental
Jogo educativo e documentação acadêmica desenvolvidos para o Projeto de Extensão: Gamificação Aplicada à Matemática do Ensino Fundamental, abordando os temas porcentagem, juros simples e juros compostos por meio de estratégias de gamificação.

## Sobre o Projeto
Este projeto foi desenvolvido no âmbito do Projeto de Extensão: Gamificação Aplicada à Matemática do Ensino Fundamental. A proposta consiste na criação de um jogo educativo voltado ao ensino de porcentagem, juros simples e juros compostos, utilizando elementos de gamificação para tornar o aprendizado mais dinâmico e interativo.

## Objetivos
- Promover o aprendizado de matemática de forma lúdica;
- Desenvolver o raciocínio lógico dos estudantes;
- Aplicar conceitos de porcentagem, juros simples e juros compostos em situações práticas.

## Estrutura do Repositório

 Relatório do Projeto de Extensão
- Documento acadêmico elaborado conforme as normas da ABNT.

Jogo Educativo
- Material principal desenvolvido para aplicação dos conceitos matemáticos.

Apostila do Professor
- Material de apoio contendo orientações pedagógicas e instruções para aplicação do jogo.

## Considerações Finais
Este projeto busca demonstrar como a gamificação pode contribuir para o ensino da matemática, tornando o aprendizado de porcentagem, juros simples e juros compostos mais acessível, interativo e significativo para os estudantes do ensino fundamental.
